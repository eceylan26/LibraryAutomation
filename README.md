# LibraryAutomation
Library Database with Java and SQL

This project has two user types. Admin and member. You can imagine that admin is a librarian and member is a student.

Admin's Permissions :

     listAllAcc()
     listAllBooks()
     deleteMember() 
     deleteBook() 
     addUser();
     addBook();
     
Member's Permissions :    

    istAllMyBooks() 
    searchBook() 
    findBookId(String bookName)
    commentBook()
    borrowBook();
    Menu();
